PHP+ Apache+PostgrSQL の設定
　　　　　　　　　　　　　　　　　　　　2008/12/17
1) PHP Ver 5.2.8 c:\php
2) Apahe 2.2.10(win32) c:\Program Files\Apache Software Fundation\PHPApache2.2
2) PostgreSQL 8.1.4. c:\c:\Program Files\PostgreSQL\8.1
上記プログラムを使っています

php.ini は下記変更しました

include_path = ".;c:\php\includes;c:\php\pear"

doc_root = "C:\Program Files\Apache Software Foundation\PHPApache2.2\htdocs"

extension_dir = "c:\php\ext"

extension=php_gd2.dll
extension=php_mbstring.dll
extension=php_pgsql.dll

mbstring.language = Japanese
mbstring.internal_encoding = EUC-JP

mbstring.http_output = SJIS
mbstring.encoding_translation = On
mbstring.detect_order = auto
mbstring.substitute_character = none;
mbstring.func_overload = 0

Apache httpd.conf

LoadModule php5_module "c:\php\php5apache2_2.dll"
AddType application/x-httpd-php .php

php -i で下記内容です

pgsql
PostgreSQL Suport => enabled
PostgreSQL(libpg) Version => 8.3.1

info.php

php version 5.2.8
Comfigure Command の欄に
--with-pgsql=
がありません

またpgsql の欄が無いです

PATH
C:\php;C:\php\ext;C:\Program Files\PostgreSQL\8.1\bin

上記PHPのバージョンとPostgrSQLのバージョンの違いで
うまく動かないでしょうか
